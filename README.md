# desafio-excel-santander-dio
Repositório para entrega do desafio de dashboard com Excel do Bootcamp Santander e DIO - Ciência de Dados com Python

Neste desafio foi proposta realizar um processo de ETL e criar um dashboard visualmente agradável a partir de uma dados de uma planilha de Excel.
Para tal, foram criadas informações fictícias de empresa e vendedores, simulando uma tabela com dados de venda por vendedor e ou por produto de uma empresa de materiais de construção, chamada Casa da Construção

O processo seguiu conforme as etapas listadas abaixo:

# Primeiro Passo
Realizar a personalização dos dados de organização ABCDE para o meu caso

# Segundo Passo
Realizar a troca de dados fictícios da tabela base para uma situação personalizada

# Terceiro Passo
Abastecer a aba de cálculos com as tabelas dinâmicas baseadas na tabela base

# Quarto Passo
Inserir os gráficos e filtros dinâmicos, para que as informações apresentadas posteriormente façam sentido

# Quinto Passo
Criar e personalizar os dashboards
